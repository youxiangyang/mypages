PostgreSQL中常用的时间函数

Posted on 周三 06 十一月 2024 in DataBase

在PostgreSQL中常用的时间函数及其使用方式和简单的SQL例子：

1. CURRENT_TIMESTAMP

返回当前日期和时间。

SELECT CURRENT_TIMESTAMP;

2. EXTRACT(field FROM source)

从日期/时间值中提取特定的部分（如年、月、日等）。

SELECT EXTRACT(YEAR FROM CURRENT_TIMESTAMP) AS year;

3. AGE(source, target)

返回source和target之间的时间差。

SELECT AGE(CURRENT_TIMESTAMP, TIMESTAMP '2000-01-01 00:00:00');

4. TO_TIMESTAMP(text, format)

将字符串转换为时间戳。

SELECT TO_TIMESTAMP('2023-01-01 14:30:00', 'YYYY-MM-DD HH24:MI:SS');

5. TO_DATE(text, format)

将字符串转换为日期。

SELECT TO_DATE('2023-01-01', 'YYYY-MM-DD');

6. NOW()

返回当前日期和时间（与CURRENT_TIMESTAMP类似）。

SELECT NOW();

7. LOCALTIME

返回当前时间，不包含时区信息。

SELECT LOCALTIME;

8. LOCALTIMESTAMP

返回当前日期和时间，不包含时区信息。

SELECT LOCALTIMESTAMP;

9. DATE_PART(text, source)

与EXTRACT类似，用于提取日期/时间值的一部分。

SELECT DATE_PART('day', CURRENT_TIMESTAMP) AS day_of_month;

10. DATE_TRUNC(text, source)

将日期/时间值截断到指定的精度。

SELECT DATE_TRUNC('hour', CURRENT_TIMESTAMP);

11. INTERVAL

用于创建时间间隔，可以与日期和时间进行加减。

SELECT CURRENT_TIMESTAMP + INTERVAL '1 day';

12. TIMESTAMP

用于创建时间戳值。

SELECT TIMESTAMP '2023-01-01 00:00:00';

这些是PostgreSQL中一些常用的日期和时间函数。每个函数都有其特定的用途，你可以根据需要选择合适的函数来处理数据库中的日期和时间数据。

PostgreSQL 提供了一系列用于时间计算的函数，以下是一些常用的函数及其用途：

时间加减

interval 'value' +

将间隔添加到时间戳。

SELECT CURRENT_TIMESTAMP + INTERVAL '1 hour';

interval 'value' -

从时间戳减去间隔。

SELECT CURRENT_TIMESTAMP - INTERVAL '2 days';

时间差计算

AGE(timestamp, timestamp)

计算两个时间戳之间的间隔。

SELECT AGE(TIMESTAMP '2023-12-31 23:59:59', TIMESTAMP '2023-01-01 00:00:00');

JUSTIFY_DAYS(interval)

调整间隔，使其天数在合理的范围内（-29 through 29）。

SELECT JUSTIFY_DAYS(INTERVAL '35 days');

JUSTIFY_HOURS(interval)

调整间隔，使其小时数在合理的范围内（-23 through 23）。

SELECT JUSTIFY_HOURS(INTERVAL '26 hours');

JUSTIFY_INTERVAL(interval)

调整间隔，使其所有字段都在合理的范围内。

SELECT JUSTIFY_INTERVAL(INTERVAL '1 year 15 months');

时间转换

TO_CHAR(timestamp, text)

将时间戳转换为字符串。

SELECT TO_CHAR(CURRENT_TIMESTAMP, 'YYYY-MM-DD HH24:MI:SS');

TO_TIMESTAMP(text, text)

将字符串转换为时间戳。

SELECT TO_TIMESTAMP('2023-01-01 00:00:00', 'YYYY-MM-DD HH24:MI:SS');

时间截断

DATE_TRUNC(text, timestamp)

将时间戳截断到指定的精度。

SELECT DATE_TRUNC('month', CURRENT_TIMESTAMP);

时间间隔函数

INTERVAL

创建时间间隔。

SELECT INTERVAL '1 day';

`MAKE_INTERVAL(years => integer, ...)

创建一个间隔，可以指定年、月、日、小时等。

SELECT MAKE_INTERVAL(days => 10);

特定时间计算

AT TIME ZONE zone

将时间戳转换为指定时区的时间。

SELECT CURRENT_TIMESTAMP AT TIME ZONE 'UTC';

CLOCK_TIMESTAMP()

返回当前实际时间（可能比CURRENT_TIMESTAMP更精确）。

SELECT CLOCK_TIMESTAMP();
