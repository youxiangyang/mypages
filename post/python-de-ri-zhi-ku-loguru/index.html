python的日志库-Loguru

2024-07-03 5 min read # python

Loguru 是一个 Python 日志库，它提供了一种简单而强大的方式来记录日志，使得日志记录变得轻松愉快。以下是 Loguru 基本使用的介绍：

快速使用

安装 Loguru

首先，您需要通过 pip 安装 Loguru：

pip install loguru

导入 Loguru

安装完成后，在您的 Python 脚本中导入 Loguru：

from loguru import logger

配置 Loguru

Loguru 允许您通过装饰器 @logger.catch 来配置日志的基本行为，例如设置日志级别、输出格式和输出位置等。例如：

# 设置日志级别为 DEBUG
logger.level("DEBUG")
# 设置日志格式
logger.format("({time:YYYY-MM-DD at HH:mm:ss}) {level} - {message}")
# 设置日志输出到文件
logger.add("myapp.log", rotation="1 week", compression="zip")
# 设置日志回滚
logger.add("myapp.log", rotation="10 MB", compression="zip", level="ERROR")

记录日志

使用 Loguru 记录日志非常简单，您可以直接调用 logger 对象的相应方法来记录不同级别的日志：

logger.debug("This is a debug message")
logger.info("This is an info message")
logger.warning("This is a warning message")
logger.error("This is an error message")
logger.critical("This is a critical message")

异常处理

Loguru 可以自动捕捉未处理的异常，并将其记录为日志：

try:
    1 / 0
except ZeroDivisionError:
    logger.exception("Attempted to divide by zero")

装饰器

Loguru 还提供了装饰器，可以用来记录函数的执行时间或捕获函数中的异常：

@logger.catch
def my_function():
    # 函数代码
    pass

@logger.timeit
def my_timed_function():
    # 函数代码
    pass

高级用法示例

Loguru 提供了许多高级用法，这些功能可以帮助您更有效地进行日志记录和管理。以下是一些高级用法的示例：

1. 自定义日志处理器

您可以使用 add() 方法来添加自定义的日志处理器，例如将日志输出到不同的文件或位置：

# 将日志输出到控制台和文件
logger.add(sys.stdout, level="INFO")
logger.add("myapp.log", level="ERROR")

2. 动态更改日志级别

您可以在运行时动态更改日志级别，以适应不同的调试需求：

# 动态更改日志级别
logger.level("WARNING")

3. 日志回滚

Loguru 支持日志文件的自动回滚，例如，当日志文件达到一定大小或一定时间后自动创建新文件：

# 每天回滚日志文件
logger.add("myapp.log", rotation="1 day")

# 达到一定大小后回滚
logger.add("myapp.log", rotation="10 MB")

4. 日志压缩

当日志文件回滚时，旧的日志文件可以被压缩，以节省存储空间：

logger.add("myapp.log", rotation="1 week", compression="zip")

5. 过滤日志

您可以使用过滤器来控制哪些日志消息应该被记录：

def my_filter(record):
    return record["level"].no >= 30  # 只记录ERROR及以上级别的日志

# 添加带有过滤器的日志处理器
logger.add("myapp.log", filter=my_filter)

6. 格式化日志

您可以自定义日志的格式，包括时间戳、日志级别、消息等：

# 自定义日志格式
logger.format(
    "[{time:YYYY-MM-DD at HH:mm:ss.SSS}] {level:<8} | {message}",
    format_time=lambda t: t.strftime("%Y-%m-%d %H:%M:%S.%f")[:-3],
)

7. 记录异常信息

Loguru 可以自动记录异常信息，包括堆栈跟踪，这对于调试非常有帮助：

try:
    # 可能引发异常的代码
    pass
except Exception as e:
    logger.opt(exception=True).error("An error occurred")

8. 使用上下文管理器

Loguru 还提供了上下文管理器，允许您临时更改日志记录的行为：

with logger.contextualize(extra=dict(component="auth")):
    logger.info("This log message will have an 'extra' field with 'component' key")

9. 性能测试

Loguru 还可以用来测量代码块的执行时间，这对于性能测试非常有用：

with logger.timeit():
    # 执行一些操作
    pass

10. 集成第三方库

Loguru 可以与许多第三方库集成，例如 Flask、Django 等，以提供更丰富的日志记录功能。

这些高级用法展示了 Loguru 的灵活性和强大功能，使其成为 Python 日志记录的强大工具。
