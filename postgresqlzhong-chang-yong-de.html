PostgreSQL中常用的

Posted on 周三 06 十一月 2024 in DataBase

PostgreSQL 提供了一系列聚合函数，这些函数可以对多行数据进行计算，并返回单个值。以下是一些常用的聚合函数：

  1. AVG(expression) - 返回表达式的平均值（算术平均）。

sql SELECT AVG(salary) FROM employees;

  1. COUNT(expression) - 返回表中行的数量，或者根据表达式计算非空值的数量。

sql SELECT COUNT(*) FROM employees; -- 计算所有行 SELECT COUNT(salary) FROM employees; -- 计算非空 salary 的行数

  1. MAX(expression) - 返回表达式中的最大值。

sql SELECT MAX(salary) FROM employees;

  1. MIN(expression) - 返回表达式中的最小值。

sql SELECT MIN(salary) FROM employees;

  1. SUM(expression) - 返回表达式的总和。

sql SELECT SUM(salary) FROM employees;

  1. ARRAY_AGG(expression) - 返回表达式的数组。

sql SELECT ARRAY_AGG(salary) FROM employees;

  1. STRING_AGG(string, delimiter) - 返回字符串的连接，使用指定的分隔符。

sql SELECT STRING_AGG(first_name, ', ') FROM employees;

  1. BIT_AND(expression) - 返回表达式位与的结果。

sql SELECT BIT_AND(permission_flags) FROM roles;

  1. BIT_OR(expression) - 返回表达式位或的结果。

sql SELECT BIT_OR(permission_flags) FROM roles;

  1. BOOL_AND(expression) - 如果所有表达式都为真，则返回真，否则返回假。

    sql SELECT BOOL_AND(active) FROM employees;

  2. BOOL_OR(expression) - 如果任一表达式为真，则返回真，否则返回假。

    sql SELECT BOOL_OR(active) FROM employees;

  3. EVERY(expression) - 如果所有表达式都为真，则返回真，否则返回假（与 BOOL_AND 类似）。

    sql SELECT EVERY(active) FROM employees;

  4. JSON_AGG(json) - 返回 JSON 对象的聚合。

    sql SELECT JSON_AGG(json_data) FROM some_table;

  5. JSONB_AGG(jsonb) - 返回 JSONB 对象的聚合。

    sql SELECT JSONB_AGG(jsonb_data) FROM some_table;

  6. MODE() - 返回最频繁出现的值。

    sql SELECT MODE() WITHIN GROUP (ORDER BY salary) FROM employees;

  7. STDDEV(expression) - 返回表达式的标准偏差。

    sql SELECT STDDEV(salary) FROM employees;

  8. STDDEV_POP(expression) - 返回表达式的总体标准偏差。

    sql SELECT STDDEV_POP(salary) FROM employees;

  9. STDDEV_SAMP(expression) - 返回表达式的样本标准偏差。

    sql SELECT STDDEV_SAMP(salary) FROM employees;

  10. VARIANCE(expression) - 返回表达式的方差。

    sql SELECT VARIANCE(salary) FROM employees;

  11. VAR_POP(expression) - 返回表达式的总体方差。

    sql SELECT VAR_POP(salary) FROM employees;

  12. VAR_SAMP(expression) - 返回表达式的样本方差。

    sql SELECT VAR_SAMP(salary) FROM employees;

聚合函数通常与 GROUP BY 子句一起使用，以对结果集进行分组，并分别计算每个组的聚合值。
